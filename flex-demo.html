Flex布局演示

该页面主要演示Flex布局中各个属性的表现效果，内容主要参考阮一峰的Flex布局教程。地址是：http://www.ruanyifeng.com/blog/2015/07/flex-grammar.html

演示中减去了基础的概念说明，包括主轴，交叉轴等，这一块不了解的可以去参考上面阮一峰的教程。

演示分两部分：第一部分放在.flexWrap中，主要说明Flex属性及取值；第二部分则放在#boxX及.itemX中，主要演示Flex属性在不同取值下的表现。

整个页面做了简单的自适应，可以通过调整页面宽度查看Flex属性的表现效果

一、 容器上的属性

flex-direction属性

这个属性是用来确定子元素的排布方向，主要分沿主轴从左到右row，沿主轴从右到左row-reverse，沿交叉轴从上到下column及沿交叉轴从下到上column-reverse。

#box1 {
	flex-direction: row | row-reverse | column | column-reverse;

flex-flow属性

这个属性是flex-direction属性和flex-wrap属性的简写形式，默认值为row nowrap。

#box {
	<flex-flow> || <flex-wrap>;
}

justify-content属性

这个属性是用来确定子元素排布在主轴上的对齐方式，包括左对齐flex-start，右对齐flex-end，中间对齐center，两侧对齐space-between及间隔对齐space-around。

二、 项目上的属性

order属性

这个属性是用来确定子元素的排列顺序，数值越小，排列越靠前。默认为0，可取负值。

.item4 {
	order: <integer>;
}

flex-grow属性

这个属性是用来确定子元素的放大比例，项目将按比例撑满宽度。默认为0，即有剩余空间也不放大。

.item7 {
	flex-grow: <number>;  /* default 0 */
}
1
2
3
/* No.1 */ { flex-grow: 0; } /* No.2 */ { flex-grow: 2; } /* No.3 */ { flex-grow: 1; }

flex-shrink属性

这个属性是用来确定子元素的缩小比例，项目将按比例缩小宽度。设置为0时，子元素不参与比例收缩。不可取负值。

.item10 {
	flex-shrink: <number>;  /* default 1 */
}
1
2
3
/* No.1 */ { flex-shrink: 1; width: 100px; } /* No.2 */ { flex-shrink: 0; width: 200px; } /* No.3 */ { flex-shrink: 1; width: 100px; }

flex-basis属性

这个属性是用来确定子元素占据主轴空间。浏览器将根据这个属性计算主轴是否有多余空间。默认值为auto，即子元素本来大小。

.item13 {
	flex-basis: <length> | auto;  /* default auto */
}
1
2
3
/* No.1 */ { flex-basis: auto; } /* No.2 */ { width: 200px; flex-basis: 200px; } /* No.3 */ { flex-basis: auto; }

flex属性

这个属性是flex-grow, flex-shrink和flex-basis的简写，默认值为0 1 auto。后两个属性可选。

#item {
	flex: none | [ <'flex-grow'> <'flex-shrink'>? || <'flex-basis'>;
}

align-self属性

这个属性是允许子元素异于其他子元素的对齐方式，直接覆盖父元素的对齐属性align-items。默认值为auto，及继承父元素对齐方式。

.item14 {
	align-self: auto | flex-start | flex-end | center | baseline | stretch;
}
